const { Telegraf } = require('telegraf'); const express = require('express'); const app = express(); // 1. Konfigirasyon Sèvè pou Render (pou l pa janm dòmi) const PORT = process.env.PORT || 3000; app.get('/', (req, res) => res.send('Qeyko Mining Pro is Online! ✅')); app.listen(PORT, () => console.log(`Sèvè ap vire sou pò ${PORT}`)); // 2. Koneksyon ak Bot la (li pral pran Token nan nan Environment Variables) const bot = new Telegraf(process.env.BOT_TOKEN); // 3. Yon ti Database senp nan memwa (pou tès la) let users = {}; // Kòmand /start bot.start((ctx) => { const userId = ctx.from.id; const username = ctx.from.first_name; if (!users[userId]) { users[userId] = { balance: 0.00, lastCheck: Date.now(), miningPower: 0.00000578 // Sa bay anviwon 0.5 USDT pa jou }; } ctx.reply(`Byenveni ${username} sou Qeyko Mining Pro! ⛏️\n\nSistèm nan kòmanse minen pou ou otomatikman.\n\nSèvi ak /balance pou wè kòb ou.`); }); // Kòmand /balance pou wè konbe kòb ki genyen bot.command('balance', (ctx) => { const userId = ctx.from.id; if (!users[userId]) { return ctx.reply("Tanpri tape /start pou w debloke kont ou."); } // Kalkile pwofi depi dènye fwa li te tcheke a const now = Date.now(); const duration = (now - users[userId].lastCheck) / 1000; // an segonn const profit = duration * users[userId].miningPower; users[userId].balance += profit; users[userId].lastCheck = now; ctx.reply(`💰 Balans ou kounye a:\n\n${users[userId].balance.toFixed(6)} USDT\n\nVitès: Standard (1x)`); }); // Kòmand /help bot.help((ctx) => { ctx.reply("Kòmand ki disponib:\n/start - Kòmanse minen\n/balance - Tcheke balans ou\n/help - Lis èd"); }); // Lansman Bot la bot.launch().then(() => { console.log("QEYKO BOT LANSE SOU TELEGRAM!"); }).catch((err) => { console.error("Erè lansman:", err); }); // Pèmèt bot la kanpe byen si sèvè a rekòmanse process.once('SIGINT', () => bot.stop('SIGINT')); process.once('SIGTERM', () => bot.stop('SIGTERM'));
